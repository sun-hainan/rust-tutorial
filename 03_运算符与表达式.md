## 1. 算术运算符与溢出处理

### 生活类比 🎯

> 算术运算就像日常计算——1+1=2 是常识，但 255+1 在 8 位世界里会"翻车"。Rust 会在 debug 模式下警告你，release 模式下默默 wrap。

### 概念讲解

Rust 的算术运算符：`+ - * / %`

**溢出行为（Rust 2018+）：**
- debug 模式：溢出触发 panic（debug assertions）
- release 模式：溢出采用二进制补码截断（wrap）

**显式控制：** 使用 `checked_*`, `wrapping_*`, `overflowing_*`, `saturating_*` 方法

### ASCII 图示

```
┌─────────────────────────────────────────────────────────┐
│              u8 溢出行为（255 + 1 = 0）                 │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  二进制:  11111111 (255)                                │
│         +00000001 (1)                                  │
│  ─────────────────                                     │
│  结果:   100000000 (256) → 截断后 → 00000000 (0)       │
│                                                          │
│  wrapping_add: 255.wrapping_add(1) = 0                   │
│  checked_add:  255.checked_add(1) = None                │
│  saturating_add: 255.saturating_add(1) = 255           │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 代码示例

```rust
fn arithmetic_operators() {
    let a: i32 = 10;
    let b: i32 = 3;

    println!("a + b = {}", a + b);
    println!("a - b = {}", a - b);
    println!("a * b = {}", a * b);
    println!("a / b = {} (整数除法)", a / b);  // 3
    println!("a % b = {}", a % b);             // 1

    // 溢出演示
    let max: u8 = 255;
    let wrapped = max.wrapping_add(1);
    println!("255 + 1 = {} (wrapping)", wrapped); // 0

    // 显式溢出检查
    let checked = 100i32.checked_add(200);
    println!("checked 100 + 200 = {:?}", checked);
}
```

## 2. 关系运算符与类型要求

### 概念讲解

关系运算符：`== != < <= > >=`

所有基本类型都实现了 `PartialEq` trait（部分相等）和 `Ord` trait（排序）。自定义类型需要 derive 或手动实现这些 trait。

### 代码示例

```rust
fn relational_operators() {
    // 数值比较
    println!("5 > 3: {}", 5 > 3);
    println!("5 == 5.0: {}", 5 == 5.0f64); // i32 和 f64 可以比较

    // 字符串比较
    let s1 = String::from("hello");
    let s2 = String::from("world");
    println!("s1 == s2: {}", s1 == s2); // false

    // 数组比较（必须元素可比较）
    let arr1 = [1, 2, 3];
    let arr2 = [1, 2, 3];
    let arr3 = [1, 2, 4];
    println!("arr1 == arr2: {}", arr1 == arr2); // true
    println!("arr1 == arr3: {}", arr1 == arr3); // false
}
```

## 3. 逻辑运算符与短路求值

### 生活类比 🎯

> 逻辑运算符的短路求值就像考试判断题——老师一看你已经错了，后面不用看了。`&&` 和 `||` 也是这样，左边能决定结果就跳过右边。

### 概念讲解

`&&` 和 `||` 是短路求值（short-circuit evaluation）：
- `&&`：左边为 false 时，不计算右边
- `||`：左边为 true 时，不计算右边

编译器视角：`&&` 和 `||` 被翻译成条件分支，不是简单的按位运算。

### 代码示例

```rust
fn logical_operators() {
    let a = true;
    let b = false;

    println!("a && b = {}", a && b); // false
    println!("a || b = {}", a || b); // true
    println!("!a = {}", !a);           // false

    // 短路求值演示
    let get_false = || { println!("evaluating false"); false };
    let get_true = || { println!("evaluating true"); true };

    println!("--- testing && ---");
    if get_true() && get_false() { /* 不打印 */ }

    // && vs & 的区别
    println!("3 && 5 = {}", 3 != 0 && 5 != 0); // true（布尔）
    println!("3 & 5 = {}", 3 & 5);             // 1（按位）
}
```

## 4. 位运算符

### 生活类比 🎯

> 位运算是"拆成零件"操作——就像检查一台机器的每个零件开关是开还是关。在权限管理、图像处理、游戏开发中非常有用。

### ASCII 图示

```
┌─────────────────────────────────────────────────────────┐
│                    位运算操作                            │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  a = 1100 (12)                                          │
│  b = 1010 (10)                                          │
│                                                          │
│  a & b = 1000 (8)  ← AND：两位都是1才为1               │
│  a | b = 1110 (14) ← OR：只要有一位是1就为1             │
│  a ^ b = 0110 (6)  ← XOR：相异为1，相同为0             │
│  ~a   = ...11110011 ← 取反                             │
│                                                          │
│  1 << 4 = 10000 (16)  ← 左移：乘以2^n                  │
│  8 >> 2 = 0010 (2)    ← 右移：除以2^n                  │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 代码示例

```rust
fn bitwise_operators() {
    let a: u8 = 0b1100; // 12
    let b: u8 = 0b1010; // 10

    println!("a = {:#06b}, b = {:#06b}", a, b);
    println!("a & b  = {:#06b} ({:3})", a & b, a & b); // 1000 (8)
    println!("a | b  = {:#06b} ({:3})", a | b, a | b); // 1110 (14)
    println!("a ^ b  = {:#06b} ({:3})", a ^ b, a ^ b); // 0110 (6)

    // 常用技巧：判断某一位是否为 1
    let flags: u8 = 0b00010101; // 21
    let bit2_is_set = (flags & (1 << 2)) != 0;
    println!("flags bit 2 is set: {}", bit2_is_set);

    // 快速乘除（位移优化）
    let num: u32 = 100;
    println!("num * 2 = {}", num << 1);
    println!("num / 2 = {}", num >> 1);
}
```

## 5. 条件表达式（无三目运算符）

### 概念讲解

Rust 没有 `?:` 三目运算符，但 `if` 是表达式，可以返回值。所有分支必须返回相同类型。

### 代码示例

```rust
fn conditional_expressions() {
    // if 作为表达式
    let condition = true;
    let value = if condition { 10 } else { 20 };
    println!("value = {}", value);

    // match 表达式（更强大的条件分支）
    let grade = 85;
    let result = match grade {
        90..=100 => "A",
        80..=89  => "B",
        70..=79  => "C",
        60..=69  => "D",
        _        => "F",
    };
    println!("grade {} → {}", grade, result);
}
```

## 本章小结

| 分类 | Rust 运算符 |
|------|------------|
| 算术 | `+ - * / %`（整数除法，溢出分 debug/release） |
| 位运算 | `& \| ^ ! << >>` |
| 逻辑 | `&& \|\| !`（短路求值） |
| 比较 | `== != < <= > >=` |
| 三目 | 无，用 `if` 表达式替代 |
| 赋值 | 返回 `()`（不支持链式） |
